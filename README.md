# OCT 视网膜影像智能诊断系统

这是一个基于 TensorFlow.js 的医学影像诊断工具，支持 DME / DRUSEN / NORMAL / CNV 的自动识别。

## 🚀 在线预览地址

*   **[版本 A] 纯前端稳定版** (点开即用): 
    [点击进入](https://medical-ai-labs.github.io/oct_classification_model/)
    
*   **[版本 B] 前后端全栈版** (需配合本地 Python 运行): 
    [点击进入](https://medical-ai-labs.github.io/oct_classification_model/index_fullstack.html)

## 🛠️ 使用说明
1. **版本 A**：直接上传图片即可诊断。
2. **版本 B**：使用前请在本地运行 `python app.py`，上传的图片会自动同步到本地 `uploads` 文件夹。
